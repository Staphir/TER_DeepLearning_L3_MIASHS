Résumé
======
Le deep-learning (sous branche du machine-learning) est une méthode de programmation d’intelligence artificielle qui se développe de plus en plus ces dernières années.
Cette méthode existe depuis longtemps mais à l’époque les technologies n’étaient pas assez puissantes. En effet, le deep-learning a besoin d’effectuer une grande quantitée de calculs.
Cette technique consiste à programmer des réseaux de neurones. Ensuite, le but est d'entraîner le programme pour qu’il apprenne et s’améliore.
Pour la programmation d’intelligence artificielle avec algorithme de deep-learning plusieurs bibliotèques existes. Mais pour notre projet nous nous sommes concentrés sur les bibliotèques utilisant le langage python
(le langage le plus utilisé pour ce type de programmation). Sur ce site vous pourrez une présentation des principales bibliotèques python permettant de faire du deep-learning ainsi que des comparaisons.
Une autre partie de notre TER est axée sur la résolution de casse-tête. C’est pourquoi une partie du site présente différentes techniques de résolutions de casse-têtes avec du deep-learning.
Enfin, nous avons commencé/fait un programme de résolution du casse-tête appelé “shiftit”. Ce casse-tête est composé d’une grille avec des carrés de couleurs différentes,
dans notre cas nous avons une grille de 5x5 avec deux couleurs (jaune et orange), le but est de créer seulement deux zones une jaune et une orange. Pour ce faire le joueur peut faire défiler chaque ligne ou colonne.
